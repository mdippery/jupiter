RichLong 

implicit class RichLong extends AnyRef

An implicit conversion from longs that adds some useful methods to long. Since ints can be promoted to longs, these methods are available to ints as well. The implicit class can be imported with:

import com.mipadi.jupiter.math.Numeric._
val isPrime = 100.isPrime
