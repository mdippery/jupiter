package colors

Provides a nice way to work with colors in the terminal.

String objects can be implicitly converted to a ColorizedString, which allows programs to print colored strings to a terminal, like so:

import com.mipadi.jupiter.io.terminal.colors._
println(s"The word ${"red".red} is red in this line.")
