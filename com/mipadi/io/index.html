    Useful classes for easily working with files and paths.

    Useful classes for easily working with files and paths.

    Provides an implicit conversion from java.util.File to RichFile, which adds some useful methods to File. It also provides an implicit conversion from java.nio.file.Path to RichPath, which provides a more convenient way to work with paths in Scala. Finally, it provides a p prefix for strings, so paths can be created directly from a string:

    import com.mipadi.io.files._
val path = p"src/main/scala"

  1.  object IO

    Wraps a function or sequence of operations in an IO object.

    Wraps a function or sequence of operations in an IO object.

    An IO object represents a function or sequence of steps that can be performed at a later time. Creating such an object is fairly simple:

    val printIO = IO { println("This is an IO operation") }

    The operation can be run by simply applying the object:

    printIO()
